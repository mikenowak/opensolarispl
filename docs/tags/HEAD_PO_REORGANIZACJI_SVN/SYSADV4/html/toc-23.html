System Administration Guide: Network Services

February 2007
This book is for anyone responsible for administering one or more systems that run the Solaris 10 release. The book covers a broad range of Solaris network administration topics such as remote file systems, mail, SLP, and PPP.Topics are described for both SPARC and x86 systems, where appropriate.
