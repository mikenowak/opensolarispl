Configuring Users and Printers for Labels

Labeling decisions need to be enforced on users, and on printers.

When setting up user accounts, the Security Administrator role needs to specify the following for every user:

  • The appropriate clearance

    To plan user clearances, see Planning the Clearances in a Worksheet.

  • The appropriate minimum label

  • Label visibility

For details, see Managing Users and Rights With Solaris Management Console (Task Map) in Solaris Trusted Extensions Administrator’s Procedures .

The Security Administrator role can customize labeling or not labeling printed output. For the procedures, see Managing Printing in Trusted Extensions (Tasks) in Solaris Trusted Extensions Administrator’s Procedures .
