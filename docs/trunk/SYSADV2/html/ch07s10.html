How the lpsched Daemon Tracks the Status of Print Requests

The lpsched daemon keeps a log file of each print request that it processes and notes any errors that occur during the printing process. This log file is kept in the /var/lp/logs/lpsched file. Every night, the lp cron job renames the /var/lp/logs/lpsched file to a new lpsched. n file and starts a new log file. If errors occur or jobs disappear from the print queue, you can use the log files to determine what the lpsched daemon has done with a printing job.
