<a name="what-pdl">
<strong>Co to jest <a href="/os/licensing/pdl/">Public Documentation License</a>?</strong>

<p>PDL to licencja Public Documentation License. Jest to oddzielna licencja dokumentacji stworzona, aby spełnić wymagania licencji, która pozwala społeczeństwie Open Source współpracować nad dokumentacją, która nie ma być włączana lub zintegrowana ze strukturą podstawową innych projektów lub dystrybucji. (np. "produkt").
</p>

<a name="what-new">
<strong>Co jest nowego w PDL v1.01?</strong>

<p><a href="/os/licensing/pdl/">PDL v1.01</a> dodaje ograniczenia używania znaków towarowych języków do <a href="http://www.openoffice.org/FAQs/faq-licensing.html#pdla">PDL v1.0</a>, które jest używane przez <a href="http://www.openoffice.org/">OpenOffice.org</a>.</p>

<a name="why-updated">
<strong>Dlaczego zostało uaktualnione?</strong>

<p>Public Documentation License v1.0 nie usuwa jasno praw znaków towarowych z praw udzielonych przez licencje. W tym celu została dodana sekcja 5.9 w PDL v1.01.  Licencja <a href="/os/licensing/opensolaris_license/">CDDL</a> ma podobną klauzulę.
</p>

<a name="what-use">
<strong>Jaki rodzaj dokumentów może używać licencji PDL v1.01?</strong>

<p>PDL v1.01 zostało stworzone aby pozwolić społeczeństwie Open Source współpracować nad dokumentacją niezależną od projektów lub produktów open source. Dla przykładu:
<ul>
 <li>Techniczna dokumentacja systemowa, nie dystrybuowana wraz z kodem,</li>
 <li>Materiały prezentacyjne wspierające projekt OpenSolaris,</li>
 <li>Dokumenty dotyczące frameworków rozwoju oprogramowania,</li>
 <li>Artykuły techniczne.</li>
</ul>
</p>
