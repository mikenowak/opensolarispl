<a name="use" id="use"><h3>Używanie OpenSolarisa</h3></a>
<p>Jeśli chcesz używać dystrybucji opartej na OpenSolarisie, wybierz jedną z dostępnych opcji: <br /><br />
<strong>Solaris Express Community Edition</strong> – DVD: <a href="/os/downloads/sol_ex_dvd/">Pobieranie</a> | CD: <a href="/os/downloads/sol_ex_cd/">Pobieranie</a></p>

<blockquote>Solaris Express Community Edition to binarne wydanie Suna dla deweloperów (nazwa kodowa &quot;Nevada&quot;). Zawiera najnowszą, nie obejmowaną pomocą techniczną wersję OpenSolarisa oraz dodatkowe technologie, które nie zostały jeszcze uwolnione do bazy źródeł OpenSolarisa. Deweloperzy mogą zbudować źródła OpenSolarisa używając tego wydania jako systemu bazowego. Uaktualniane co drugi piątek.</blockquote>

<p><strong>Solaris Express Developer Edition</strong> - <a href="http://www.sun.com/software/solaris/solaris-express/get.jsp">Pobieranie</a> | <a href="http://www.sun.com/software/solaris/solaris-express/">Więcej informacji</a></p>
<blockquote><p>Solaris Express Developer Edition to oficjalne wydanie Suna zawierające OpenSolarisa oraz dodatkowe technologie, które nie zostały jeszcze uwolnione do bazy źródeł OpenSolarisa. Sun oferuje ograniczoną pomoc techniczną do tego wydania. Głównym celem jest umożliwienie klientom Suna zapoznania się z ostatnimi nowinkami, które ostatecznie trafią do produktów Suna. Wydanie jest uaktualnianie co miesiąc i można je pobrać za darmo w wersji binarnej.</p></blockquote>

<strong>BeleniX</strong> – <a href="http://www.genunix.org/distributions/belenix_site/download.html">Pobierz</a> | <a href="http://www.genunix.org/distributions/belenix_site/belenix_home.html">Więcej informacji</a>

 <blockquote>BeleniX jest dystrybucją *NIX-a opartą na źródłach OpenSolarisa. Obecnie jest dystrybucją typu LiveCD, ale w zamiarze ma stać się kompletną dystrybucją do instalacji na dysku twardym.</blockquote>

 <strong>marTux</strong> – <a href="http://www.martux.org/RELEASES/">Pobierz</a> |  <a href="http://www.martux.org/READ-ME.ascii">Więcej informacji</a>
 <blockquote>marTux to pierwsza dystrybucja dla platformy SPARC (na razie sun4u, później również sun4v) inna niż Solaris Express/Solaris Express Community Release OpenSolaris.</blockquote>

 <strong>NexentaOS</strong> – <a href="http://www.gnusolaris.org/Download">Pobierz</a> | <a href="http://www.gnusolaris.org/gswiki">Więcej informacji</a>
 <blockquote>NexentaOS to dystrybucja w całości oparta o system GNU zawierająca OpenSolarisa. NexentaOS integruje OpenSolarisa (jądro SunOS) oraz aplikacje Open Source Software (OSS) - system, który wybiera najlepsze elementy z obydwu.</blockquote>

 <strong>SchilliX</strong> – <a href="http://www.genunix.org/distributions/schillix/ftp/">Pobierz</a> |  <a href="http://www.genunix.org/distributions/schillix/index.html">Więcej informacji</a>
 <blockquote>SchilliX to Uniksowy LiveCD oparty o OpenSolarisa dla platform x86, x64 i EM64T. Obecne wydania oparte są na Nevada Build 17 i późniejszych.</blockquote>

<a name="build" id="build"><h3>Budowanie OpenSolarisa</h3></a>
 <p>Projekt OpenSolaris składa się z wielu różnych komponentów programowych, z których większe nazywamy  &quot;konsolidacjami&quot;. Konsolidacja System Operacyjny/Sieć (Operating System/Networking: ON) tworzy bazę dla OpenSolarisa. Aby zbudować OpenSolarisa ze źródeł, należy najpierw zainstalować odpowiednią dystrybucję OpenSolarisa; obecnie wybór ograniczony jest do Solaris Express Community Release (powyżej).</p>

 <h4>Konsolidacje OpenSolarisa</h4>
 <p>Projekt OpenSolaris składa się z wielu różnych komponentów programowych, z których większe nazywamy  &quot;konsolidacjami&quot;. Konsolidacja System Operacyjny/Sieć (Operating System/Networking: ON) tworzy bazę dla OpenSolarisa.</p>
 <a href="/os/downloads/on/">Operating System/Networking (ON)</a>
 <blockquote>ON zawiera jądro, usługi sieciowe, biblioteki systemowe i komendy.</blockquote>
 <p><strong>Dodatkowe konsolidacje OpenSolarisa:</strong></p>

 <a href="https://glassfish.dev.java.net/">Application Server</a>
 <blockquote>Sun Java System Application Server.</blockquote>

 <blockquote>Konsolidacja DevPro zawiera różnorakie komponenty: systemowa biblioteka matematyczna, biblioteka mediów, biblioteka mikrozadań, SCCS i make oraz biblioteki czasu wykonania C++. Niektóre dostępne są jako źródła, inne jako binaria.</blockquote>
 <a href="/os/downloads/docs">Dokumentacja (Docs)</a>
 <blockquote>Dokumentacja techniczna dla deweloperów i administratorów. Dostępne są źródła wielu książek. Szczegóły na stronie pobierania.</blockquote>

 <a href="/os/community/int_localization/sources/">Obsługa międzynarodowości (G11N)</a>

 <a href="/os/downloads/install">Obsługa instalacji (Install)</a>
 <blockquote>Obsługa instalacji i narzędzi pakietów.</blockquote>
 <a href="/os/community/desktop/communities/jds/">Java Desktop (JDS)</a>
 <blockquote>Bezpieczne i kompletne graficzne środowisko pracy.</blockquote>

 <a href="http://jdk-distros.dev.java.net/">Java Platform, Standard Edition (Java SE)</a>
 <blockquote>Binaria Java Development Kit (JDK) oraz Java Runtime Environment (JRE).</blockquote>
 <a href="/os/downloads/manpages">Strony man</a>
 <blockquote>Źródła stron man do SunOS.</blockquote>
 <a href="http://mq.dev.java.net/">Kolejka wiadomości</a>

 <a href="/os/community/storage/">Network Storage (NWS)</a>

 <a href="/os/downloads/sfw/">SFW (Solaris FreeWare)</a>

 <a href="/os/downloads/sparc_graphics">Obsługa grafiki na platformie SPARC</a>
