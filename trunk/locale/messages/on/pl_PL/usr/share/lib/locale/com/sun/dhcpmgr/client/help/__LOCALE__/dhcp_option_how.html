So wird's gemacht: Optionen

Anzeigen von Optionen

Wählen Sie im Hauptfenster die Registerkarte Optionen.

Der DHCP-Manager zeigt eine Liste der Optionen an, die in dhcptab des DHCP-Servers enthalten sind.

Erstellen einer neuen Option

  1. Wählen Sie im Hauptfenster die Registerkarte Optionen.
    Der DHCP-Manager zeigt eine Liste der Optionen an, die in der DHCP-Datenbank enthalten sind.

  2. Wählen Sie Erstellen aus dem Menü Bearbeiten.
    . Das Dialogfeld "Option erstellen" wird geöffnet.

  3. Geben Sie die gewünschten Einstellungen für die neue Option ein.
    Sie können die folgenden Einstellungen im Dialogfeld "Option erstellen" eingeben:
    • Optionsname
    • Kategorie
    • Client-Klasse
    • Code
    • Datentyp
    • Granularität
    • Maximum

  4. Klicken Sie auf "OK", um Ihre Einstellungen zu bestätigen.
    Die neue Optionsdefinition wird in dhcptab gespeichert und kann nun in ein Makro aufgenommen werden.

Duplizieren einer Optionsdefinition

  1. Wählen Sie im Hauptfenster die Registerkarte Optionen.
    Der DHCP-Manager zeigt eine Liste der Optionen an.

  2. Wählen Sie die zu duplizierende Optionsdefinition.

  3. Wählen Sie Duplizieren aus dem Menü Bearbeiten.
    . Das Dialogfeld "Optionseigenschaft duplizieren" wird angezeigt. Die Felder zeigen die Einstellungen der gewählten Option.

  4. Geben Sie einen Namen für die neue Option ein.

  5. Ändern Sie die Einstellungen der Optionsdefinition wie gewünscht, und klicken Sie auf "OK".
    Die neue Optionsdefinition wird in dhcptab gespeichert.
Ändern einer Optionsdefinition

  1. Wählen Sie im Hauptfenster die Registerkarte Optionen.
    Der DHCP-Manager zeigt eine Liste der Optionen an.

  2. Wählen Sie die zu ändernde Optionsdefinition.

  3. Doppelklicken Sie auf die Option, oder wählen Sie Eigenschaften aus dem Menü "Bearbeiten".
    Das Dialogfeld "Eigenschaften für Option" wird angezeigt.

  4. Ändern Sie die Einstellungen der Optionsdefinition wie gewünscht, und klicken Sie auf "OK".
    Die neuen Einstellungen werden in dhcptab aktualisiert.
Löschen einer Optionsdefinition

  1. Wählen Sie im Hauptfenster die Registerkarte Optionen.
    Der DHCP-Manager zeigt eine Liste definierter Optionen an.

  2. Wählen Sie die zu löschende Optionsdefinition.

  3. Wählen Sie Löschen aus dem Menü Bearbeiten.
    . Die Optionsdefinition wird aus der Datenbank dhcptab gelöscht.
