Dekonfigurieren des BOOTP-Service

Klicken Sie auf "OK", um den BOOTP-Relay-Service für diesen Server zu dekonfigurieren. Der Server hört auf, Anforderungen für die DHCP-Konfiguration weiterzugeben. Um den BOOTP-Service wiederherzustellen, müssen Sie den Server mit dem Dialogfeld "BOOTP-Konfiguration" erneut konfigurieren.

Verwenden Sie die Option Anhalten oder Deaktivieren aus dem Menü "Service" um den DHCP-Service anzuhalten oder zu deaktivieren.
