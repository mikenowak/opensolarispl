Löschen von Netzwerken

In der Liste "Netzwerke beibehalten" werden die Netzwerke angezeigt, die für diesen Server konfiguriert sind. Wenn Sie ein Netzwerk löschen, wird dieses aus der Liste der Netzwerke für diesen Server entfernt und damit nicht mehr vom Server überwacht.

Verschieben Sie ein Netzwerk mithilfe der Pfeile in die Liste "Netwerke löschen".

Hosts-Tabelleneinträge löschen: Aktivieren Sie dieses Feld, um alle Hostnamen, die den gelöschten Netzwerken zugeordnet sind, aus der Tabelle hosts zu löschen.

Wenn Sie ein Netzwerk zur Liste der konfigurierten Netzwerke hinzufügen oder ein gelöschtes Netzwerk wiederherstellen möchten, müssen Sie das gewünschte Netzwerk mit dem Netzwerkassistenten konfigurieren.
