Wenn "ZFS File System Management" in der Spalte "Eingeschlossene Rechte" enthalten ist, besteht die Berechtigung, Befehle für die Verwaltung von ZFS-Dateisystemen zu verwenden. Darunter fällt das Erstellen und Löschen von Dateisystemen, das Erstellen von Momentaufnahmen und Klonen, das Abrufen und Festlegen von Eigenschaften sowie das Einhängen und Aushängen von Dateisystemen.

Wenn "ZFS File System Management" grau hinterlegt ist, sind Sie nicht berechtigt, dieses Recht hinzuzufügen oder zu entfernen.
